# 大阪公立大学 過去問演習アプリ

## 概要
本アプリは、受験予定者が受験準備のために使用する、過去問演習用のウェブアプリケーションです。

## 注意事項（重要）
入試問題は受験予定者が受験の準備に使用することや、教育機関（営利目的の機関は含みません。）の教職員が教育の一環として使用することを目的としています。
本アプリ上のコンテンツ（入試問題の複製等）について、それ以外の目的で複製、転載、転用することを固く禁止します。
また、入試問題を二次利用する場合は、別途著作権許諾処理等を行ってください。

## 出典
本アプリに含まれる問題データは、大阪公立大学が公開している入学試験問題に基づいています。

## 免責事項
本アプリは個人の学習用として作成されたものであり、大阪公立大学の公式なツールではありません。
